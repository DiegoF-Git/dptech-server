Project Proposal

Today, households utilize their computers for work‑from‑home employment, online study and entertainment. When a machine goes down, downtime means lost productivity and unnecessary stress. DP IT Technology can provide affordable Level 1 support for Windows and macOS laptops and desktops in the Perth metropolitan area to fulfill this need. DP IT Technology's core services include performing operating‑system reinstallation and imaging, malware removal, peripheral installation / configuration, software installation, basic hardware cleaning and remote troubleshooting via secure screen sharing.

DP IT Technology's project website is hosted on an Ubuntu 22.04 LTS EC2 instance and delivers three key functions:

  1. Displays bundled services with clear pricing — Bronze (one fix), Silver (annual check‑up) and Gold (priority support).
  2. Uses the WP Simple Booking “Book Now” form to schedule on‑site visits or remote sessions; occupied slots are automatically removed until a new time window is opened.
  3. Provides a knowledge‑base blog with step‑by‑step instructions for common issues such as “PC won’t boot after Windows Update” or “How to recover files from a corrupted USB drive”, establishing company credibility.

Technical stack: Apache 2.4, PHP‑FPM 8.1, MariaDB 10.6 and WordPress 6.x running the lightweight GeneratePress theme, secured with a Let’s Encrypt TLS certificate. Security hardening follows AWS Well‑Architected guidelines: (i) SSH allowed only from a single IP, (ii) unattended OS updates, (iii) daily encrypted EBS snapshots and (iv) CloudWatch uptime alarms.

Project KPIs are: (i) page‑load time < 2 s for Perth visitors, (ii) 100 % uptime during business hours (Mon–Sat 08:00–18:00), (iii) ten qualified leads per month by week 8 and (iv) a site‑visit‑to‑booking conversion rate of ≥ 20 % by semester‑end. Completion of this project will demonstrate practical ability to deploy cloud‑based infrastructure, secure Linux servers and deliver real‑world IT services, fully aligned with ICT171 learning outcomes.
